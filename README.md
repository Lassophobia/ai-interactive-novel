# 天衍万象

天衍万象是一款基于 DeepSeek API 的本地 AI 互动小说创作工具。用户可以设定世界观、角色和故事风格，再通过选项或自定义行动推动剧情发展。

## 主要功能

- 自定义世界观、主角、故事背景和写作风格
- AI 自动生成剧情正文和分支选项
- 支持输入自定义行动，不受固定选项限制
- 连续章节生成和上下文衔接
- 人物关系、角色状态和世界状态追踪
- 本地存档、读档以及 JSON 导入导出
- 多种阅读主题、字号和排版设置
- 单文件 HTML，无需安装即可使用

## 使用方法

1. 下载仓库中的 `index.html`。
2. 使用现代浏览器打开该文件。
3. 进入设置页面，填写自己的 DeepSeek API Key。
4. 创建世界设定和角色后开始故事。

## 数据与隐私

- API Key 只保存在当前浏览器的本地存储中。
- 故事存档默认保存在浏览器本地。
- 仓库不包含任何 API Key、用户存档或生成内容。
- 建议定期使用 JSON 导出功能备份重要故事。

## 技术说明

项目采用单文件 HTML 结构，界面、样式和主要逻辑均包含在 `index.html` 中，可以直接通过 `file://` 方式运行。
